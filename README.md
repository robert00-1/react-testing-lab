# Title
 The  Royal Bank of Flatiron

 # Description

A React project to practice component rendering, state management, and testing using Vitest. The project simulates a small transaction manager with search, sort, and add transaction functionality.

# Tools
React
Javascript
Vites
Node.js&npm

 # Features

Display a list of transactions

Search transactions by description

Sort transactions by amount, description, or category

Add new transactions

Fully tested with Vitest

# Usage

View Transactions: The app shows a default list of transactions on load.

Search Transactions: Type in the search box to filter transactions by description.

Sort Transactions: Use the sort dropdown to order transactions by amount, description, or category.

Add Transactions: Use the form to append a new transaction to the list.
